\section{Microcontroller}
The requirements for the Microcontroller are:\\
\begin{itemize}
    \item Cortex-M-Series
    \item Security features like a TrustZone
\end{itemize}
\section{Interfaces}
The requirements for the interfaces are:\\
\begin{itemize}
    \item I2C
    \item SPI
    \item UART
    \item SWD
    \item VCOM
    \item USB
\end{itemize}

\section{Periphery}
The requirements for the peripherals are:\\
\begin{itemize}
    \item Buttons
    \item Dip-Switches
    \item LEDs 
    \item Display
    \item Pin-header for external hardware\\
\end{itemize}
